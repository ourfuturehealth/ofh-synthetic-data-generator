# Raw Data

Store public source artefacts here, copied exactly as received. Do not edit files in this
directory; if a file needs cleaning, parsing or normalization, write the result to
`data/processed/`.

Expected filenames:

- `Our Future Health Baseline Questionnaire Logic v2.2.xlsx`
- `our_future_health_data_dictionary_v14.xlsx`
- `our_future_health_codings_v14.xlsx`

Source locations:

- Data dictionary and codings:
  `https://research.ourfuturehealth.org.uk/data-and-cohort/`
- Questionnaire logic:
  `https://ourfuturehealth.gitbook.io/our-future-health/data-types/questionnaire-data`

Source context:

- Release target: P14.
- Questionnaire target: baseline health questionnaire V2.
- Download/access date: 11 June 2026.

These source artefacts are included to support reproducible registry builds. Generated
registries remain under `data/processed/`, and generated synthetic outputs are written under
`outputs/synthetic/`. Generated artefacts are ignored by git by default.
